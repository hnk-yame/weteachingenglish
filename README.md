# 🎮 We Teaching English - Kelime Yarışması

5. sınıf 1. ünite İngilizce kelimeleriyle 2 oyuncunun online yarışabileceği eğitim oyunu.

## 🎯 Oyun Kuralları

- **2 Oyuncu**: Online olarak birbirine karşı yarışır
- **4 Kelime**: Her turda 4 İngilizce kelime yazılması gerekir
- **Puan Sistemi**:
  - ✅ Doğru cevap: +10 puan
  - ❌ Yanlış cevap: -2 puan
- **Oda Kodu**: Oyuncular oda kodu ile bağlanır

## 📚 Kelime Kategorileri

- Family (Aile)
- Numbers (Sayılar)
- Colors (Renkler)
- School (Okul)
- Animals (Hayvanlar)

## 🚀 Nasıl Başlarım?

### 1. Kurulum
```bash
git clone https://github.com/hnk-yame/weteachingenglish.git
cd weteachingenglish
pip install -r requirements.txt
```

### 2. Oyunu Başlat
```bash
python server.py
```

### 3. Tarayıcıda Aç
http://localhost:5000

### 4. Oyunla

1. **Oyuncu 1**: Adını gir, oda kodu oluştur (örn: ABC123) → "Oda Oluştur"
2. **Oyuncu 2**: Adını gir, aynı oda kodunu gir → "Odaya Katıl"
3. Oyun otomatik başlar
4. 4 kelimeyi sırasıyla doğru yazarak puan kazanın
